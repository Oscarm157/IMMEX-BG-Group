## BMS Customs System (software socio)
- Títulos: "BMS Trade Compliance Software" / "Customs Compliance and Risk Control Software".
- Objetivo: **automatizar procesos y operaciones de comercio exterior**.
- Software personalizable para clientes multinacionales y multirregionales; disponible en escritorio, web y app.
- Capacidades:
  - Cumple **Anexo 24** (módulos regulatorios incluidos)
  - **Archivos EDI** (conexión electrónica vía web services)
  - **+100 reportes** de operación + Crystal Report
  - **Integración MRP/ERP** (hasta 10 transacciones automáticas)
  - Cumple **Anexo 30** (creación de documentos electrónicos)
  - Módulo **TLCAN / T-MEC** (soporte CF28)
  - **Licencias** en varios modelos (on-demand, 1–150 usuarios)
  - Proceso de **análisis y backflush** de alto rendimiento
- Métricas: **150+** usuarios soportados, **35** bases de datos por RFC, **50,000** operaciones import-export, **15M+** líneas en logs de backflush, comunicación electrónica y alertas de almacén.
